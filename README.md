# 🤖 Prácticas y Talleres de Inteligencia Artificial  

Bienvenido a este repositorio, donde se recopilan mis **prácticas, ejercicios y talleres** relacionados con el aprendizaje y aplicación de **Inteligencia Artificial (IA)**.  
El objetivo es documentar de manera organizada los temas estudiados, los códigos implementados y los resúmenes teóricos para facilitar el aprendizaje y la consulta futura.  

---

## 📌 Objetivos del repositorio
- Consolidar conocimientos fundamentales de **Python, NumPy, R y técnicas de IA**.  
- Desarrollar ejemplos prácticos y reproducibles para estudiantes y profesionales.  
- Crear material didáctico en formato **Markdown** para uso en Google Colab y entornos educativos.  
- Facilitar la comprensión mediante **tablas comparativas, resúmenes rápidos y ejercicios guiados**.  

---

## 🛠️ Tecnologías utilizadas
- **Lenguajes:** Python, R  
- **Bibliotecas:** NumPy, Pandas, Matplotlib, Scikit-learn  
- **Entornos:** Google Colab, Jupyter Notebook
